DEGENSWRITE CORPORATION

UI refinements, fixes to themes, new Settings options, a new look for Library, improved degenswrite Home and more!

Welcome to the next weekly incremental set of updates to degenswrite, the 0.9.2.6. Read on, for the full changelog.

Changes to degenswrite Home

- Added on-hover animations to tiles on home page. Hover to see them enlarge and become small.


- Adapted home tiles to their narrower variations.


- Redesigned "UPDATES" section. Now complies with alphaUI 1.4 (NORDIC) and guidelines.

Changes to Settings

- Refinements under Settings page UI to texts and sub-menus.

NEW SETTING: Adjust UI Scaling
- Added option to adjust UI-Scaling across site (incl. title pages).
- Change sizes (small/ standard/ large) from the Settings page.


- Added "Show Tips", "UI Scaling" and "Enter Support Key" to Settings page search index.

- Moved "special features" to just before "degens lab".

- Fixed light mode on Settings.

- Removed remaining indicators for the option to disable alphaUI design since alphaUI is now globally enforced from update 0.8.x.x onwards.

- Split Settings options under "display and appearance" to "themes and appearance" and "display preferences".

Changes to Library
(Previously Your Library)

- Library menu tabs get a new sleeker look!


- Library is now fully compliant and works with light mode.

- Renamed "YOUR LIBRARY" to "LIBRARY".
- Removed inner headings when Library items were opened since they already exist in buttons.

Content and Titles

- Changed write-up for "Author's note" for "The Night".

Others

- Fixed issue where the navbar link collapse button won't work on (your) Library.
- Fixed issue where the navbar link collapse button won't work on the Settings page.
- Fixed light mode not working on page change.

Incremental Changes:
- Globally incremented release ID and build details.
- Added latest changelog link to all placeholders.
- Furnished latest changelog on blog.degenswrite.com
