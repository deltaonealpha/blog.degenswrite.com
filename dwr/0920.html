DEGENSWRITE CORPORATION

Updated design with dynamic headers, automatic bookmarks which retain your position on each page, unlimited reading list imports, new daily streaks, new sorting options, "scroll-for-me" and much more!

Welcome to the next monthly iteration of updates to degenswrite, the 0.9.2.0. Read on, for the full changelog.

New Daily Continuous Streaks on Home Page

The home page now has a new daily-continuous streaks system. Your streak counts are shown along with the days of the week when you checked in, with green ticks for days checked-in on, and white for the rest. A button is given to register your streak for the day. If you've already checked in, a message will be shown and the button will change its text.

A special message is shown on the 69th day ;)

Changes to Home Page

- Added DAILY CONTINOUS STREAKS on the home page.

- Fixed light mode didn't apply on the home page.

- The home-page header now says "Good Evening/ Morning/ Afternoon"

- Good Evening/ Morning/ Night/ Afternoon on header.

- Removed the "CONTACT" link from the navbar on all pages.

- Moved search downwards (towards the bottom of the cover) on all pages with degenswriteSearch. Changed search bar and "ADVANCED" button colours to #1b212a from #181818a6.

Automatic Bookmarks

Doing away with manual bookmarks (add/ use), we are now introducing automatic bookmarks which save your position on each page.

Your position on each title page is auto-saved every 0.2 seconds and is immediately restored upon re-opening the title.

Unlimited Reading List Imports

Presenting Reading Lists 3.0
You can now import an unlimited number of reading lists to your degenswrite account. Implemented a new version of "MANAGE IMPORTED READING LISTS" on Settings to allow you to delete all imported reading lists individually.

- Now also shows your two primary reading lists with options to clear them each.

- Imported reading lists are now shown AFTER the two primary (stories/ poems) reading lists.

- Imported RLs on your library now show the inbuilt codenames, on-screen as "(ID: )".

"Scroll-for-me"

Introducing "Scroll-for-me", an option that scrolls through the title page for you. You can choose between 3-speed settings, and stop the auto-scroll feature via the options displayed on the same page.

More Sorting Options

You can now sort by flavours. The current implementation only includes a functional placeholder. The function will be enabled once we have more content.

Updated Design w/ Dynamic Headers

- Reduced navbar height on all pages by 10px (30px >> 10px).

- Added dynamic navbar brandings to settings, community, about, alphaUI, poems, stories, whatsnew pages. The navbar branding changes to the header on the page once you scroll past it (the header). The navbar branding is restored once you scroll above the header (once in view).

Others

- Fixed scroll indicator accuracy by 2-3x on title pages.

- Renamed "Pageflight and Caching" to "special features".

- Removed float from the "JUMP TO BOOKMARKED LOCATION" button.

- Fixed issue where tapping on a bookmark from your library > bookmarks lead to the 404: NOT FOUND page. Fixed it.

- On settings, patched issue where buttons turned white when their collapsible was expanded.

- Changed "you can now import one rl" to "unlimited rls" on your_library/rls, and in settings.

- Decreased Profile account name font size from 30 px >>> to >>> 21 px to better accommodate longer names on mobile devices, and allowed to set account name of 20 chars up from the earlier 15 chars.

- Removed redundant reading_list, dephased_profile pages from the distribution.- Removed older streaks implementation "OVERALL STREAK RECORD" From Profile > reading insights > "VIEW MORE".
