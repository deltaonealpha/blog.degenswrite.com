DEGENSWRITE CORPORATION

A new home-page design, better continue, delete bookmarks, dynamic floating toolbar, paper textures on settings and a shit-ton of fixes.

Welcome to the next monthly incremental set of updates to degenswrite, the 0.9.4.0. Read on, for the full changelog.

New Home-Page Design

With 0.9.4.0-CRANIEWR we take a content-first approach.

- Content-first approach.
- Home cover banner made minimal.
- Removed "Advanced Search"
- Reduced height of autobookmark tile from 150px >> 111px.

Better Continue and Progress Tracking

- "CONTINUE WHERE YOU LEFT-OFF" when progress < 90%
- "TITLE LAST READ" when title progress > 90% 
(i.e. reader is on author's note, comments, poll, footer sections and not on content)


Delete Automatic Bookmarks

- Allowed to delete automatic bookmarks from Library now.

Dynamic Floating Toolbars

- Submenus now collapse if another one is opened.
- Replaced time on floating toolbar with a link to the home page.

Paper Textures on Settings

- Enable/ disable paper textures via settings now.

Others

- Removed "WRITING PROMPTS" from home tiles.
- Settings headers are now first-char-caps and purple.
- Added discord invite link tile to home and community pages.

Renames:
- "READING LIST" to "LIBRARY" on home
- "SCROLL THIS PAGE FOR ME" to "ENABLE"
- "MAKE PAGE LOOK LIKE PAPER" to "ADD PAPER TEXTURES"
- "DISABLE PAPER-LIKE TEXTURES" to "REMOVE PAPER TEXTURES".

Fixes and Changes:
- Title tiles now shown at 90% width.
- Fixed reading list mismatch while adding to them.
- Increased border curves for nature_sounds section.
- Fixed - nature sounds collapsible didn't open.
- Reduced the size of "Screensaver is active" text.
- Fixed white mode didn't work on title pages without a reload.
- Patched issue where "SCROLL THIS PAGE FOR ME" didn't work.
- Navbar now translucent on title-pages.
- Removed "Predictive Preferences and Adaptive Settings" from settings page.
- Fixed issue with streaks where the streak count and "DAY"/ "DAYS" didn't have a spacer in between.

Incremental Changes:
- Globally incremented release ID and build details.
- Added latest changelog link to all placeholders.
- Furnished latest changelog on blog.degenswrite.com
